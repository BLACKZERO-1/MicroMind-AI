## 🌟 About

MicroMind AI is an intelligent, AI-powered microbiology assistant built by a real microbiologist. Unlike generic AI tools, MicroMind AI is specialized exclusively for life sciences — built by someone with real laboratory experience at the **National Institute of Health (NIH) Islamabad**, **Alpha Genomics**, and **NARC**.

It combines domain expertise with modern AI to provide instant, accurate, and practical microbiology guidance for students, lab technicians, and researchers.

| Module | Description |

## 🚀 Getting Started

### Prerequisites
- Python 3.11+
- Groq API key (free at console.groq.com)

### Installation

```bash
# Clone the repository
git clone https://github.com/BLACKZERO-1/MicroMind-AI.git
cd MicroMind-AI

# Install dependencies
pip install -r requirements.txt

# Create .env file
echo "GROQ_API_KEY=your_key_here" > .env

# Run the app
streamlit run app.py
```
